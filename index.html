DECIMAG

Execution Legitimacy Infrastructure

A missing layer between authorization and execution.

Most systems verify whether an action can be performed. DECIMAG evaluates whether it should be performed.

Authentication Authorization Execution Legitimacy Execution

A successful login does not prove a legitimate action. A valid permission does not prove a legitimate action.

Right-to-Act

Right-to-Act is a pre-execution decision model. It asks whether enough legitimacy exists for an action to proceed before real-world consequences occur.

Can Act

Identity passed. Permission exists. The system can execute.

Should Act

Purpose, scope, authority, consequence and reversibility are evaluated before execution.
